Everything you need, nothing you don't

A focused editor with Claude Code wired into the core.

Claude Code built in

The real claude CLI in a side panel — no API key, signed in like your terminal.

GitHub source control

Commit, sync, branch and sign in to GitHub — a clean SCM panel that knows your repo's state.

Themes & palettes

Built-in themes, a full color editor, and saved palettes — synced to your account.

Terminal & split editing

An integrated shell, split panes, smooth-caret editor, and project switching with custom icons.

Account sync

Sign in and your settings, palettes and fonts follow you to every machine.

Tiny & fast

Built on Tauri — a few megabytes, native blur, instant startup. Not another Chromium.

🪟 Windows

Download and run the installer. On the SmartScreen prompt choose More info → Run anyway.

Anode.exe

🍎 macOS

Build from source, then right-click → Open (unsigned). Or:

xattr -cr /Applications/Anode.app

🐧 Linux

Grab the AppImage and run it:

chmod +x Anode.AppImage
